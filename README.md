# 🔍 Smart Job Scraper (Internshala)

A Python automation tool that scrapes job listings from Internshala, filters them based on user-defined keywords, and exports the results into a structured CSV file.

---

## 🚀 Features

- 🔄 Scrapes real-time job listings using Selenium
- 📜 Handles dynamic content with automated scrolling
- 🧠 Keyword-based filtering (supports multiple keywords)
- 📊 Extracts:
  - Job Title
  - Company Name
  - Location
- 🧹 Removes duplicate results
- 💾 Saves data into CSV with date-based filename

---

## 🛠️ Tech Stack

- Python
- Selenium WebDriver
- CSV
- Basic Web Scraping & Automation
